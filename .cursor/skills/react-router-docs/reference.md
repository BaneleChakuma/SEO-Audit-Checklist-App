## Components (declarative)

| Component | Purpose |
|-----------|---------|
| `BrowserRouter` | History-based router for web SPAs |
| `Routes` / `Route` | Declarative route config |
| `Outlet` | Renders matched child route |
| `Link` | In-app navigation link |
| `NavLink` | Link with active state |
| `Navigate` | Declarative redirect |

Import from `react-router-dom` (v6) or `react-router` (v7+). See SKILL.md import table.

## Hooks (common)

| Hook | Purpose |
|------|---------|
| `useNavigate` | Programmatic navigation |
| `useParams` | Dynamic URL segments |
| `useLocation` | Current location object |
| `useSearchParams` | Query string read/write |
| `useMatch` | Match current URL to a pattern |
| `useOutletContext` | Pass data from layout to child routes |

Full hook list: https://api.reactrouter.com/v8/modules/react_router.html

## Patterns relevant to this app

| Pattern | Approach |
|---------|----------|
| Auth guard | Wrapper component + `useAuth` + `<Navigate replace />` |
| Post-login redirect | `navigate(from, { replace: true })` with `location.state` |
| Shared Navbar/Footer | Layout route with `<Outlet />` |
| 404 page | `<Route path="*" element={<NotFoundPage />} />` last |
| Netlify SPA | `/* /index.html 200` in `_redirects` |
